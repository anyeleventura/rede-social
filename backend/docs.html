EndPoints de Usuários

POST /usuario

Cria um novo usuário.

{
  "nome": "Nome do usuário",
  "email": "email@dominio.com",
  "senha": "senha123"
}

Resposta de sucesso:

{
  "message": "Usuário criado com sucesso!",
  "user": {
    "objectId": "12345",
    "nome": "Nome do usuário",
    "email": "email@dominio.com",
    "senha": "senha123"
  }
}

GET /usuario

Retorna todos os usuários cadastrados.

[
  {
    "objectId": "12345",
    "nome": "Nome do usuário",
    "email": "email@dominio.com",
    "senha": "senha123"
  }
]

GET /usuario/:id

Retorna um usuário específico pelo ID.

{
  "objectId": "12345",
  "nome": "Nome do usuário",
  "email": "email@dominio.com",
  "senha": "senha123"
}

PUT /usuario/:id

Atualiza os dados de um usuário.

{
  "nome": "Novo Nome",
  "email": "novo-email@dominio.com",
  "senha": "nova-senha123"
}

Resposta de sucesso:

{
  "message": "Usuário atualizado com sucesso!",
  "user": {
    "objectId": "12345",
    "nome": "Novo Nome",
    "email": "novo-email@dominio.com",
    "senha": "nova-senha123"
  }
}

DELETE /usuario/:id

Deleta um usuário pelo ID.

{
  "message": "Usuário deletado com sucesso!"
}

EndPoints de Amizade

POST /amizade

Cria uma nova amizade entre dois usuários.

{
  "usuario_id_1": "12345",
  "usuario_id_2": "67890"
}

Resposta de sucesso:

{
  "message": "Amizade criada com sucesso!"
}

GET /amizade/:id

Retorna detalhes sobre a amizade entre dois usuários pelo ID.

{
  "amizade_id": "12345",
  "usuario_id_1": "12345",
  "usuario_id_2": "67890",
  "status": "aceita"
}

DELETE /amizade/:id

Deleta uma amizade pelo ID.

{
  "message": "Amizade deletada com sucesso!"
}
